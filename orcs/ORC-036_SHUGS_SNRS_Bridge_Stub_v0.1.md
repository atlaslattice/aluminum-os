## §0 Problem Statement

Two independent expressions of the "144-sphere architecture" exist within the ORC ecosystem:

1. **SHUGS Lattice** (WP-004, eternal/resonance layer) — The Sovereign Holistic Unified Governance System. Describes the 144 spheres as resonance domains with eternal/metaphysical properties. Authored primarily through Gemini S2 and early Council sessions.

2. **SNRS 144+1** (operational governance layer) — The Sovereign Node Routing Standard. Describes the same 144 spheres as operational routing domains with module assignments, doctrine governance, and invariant enforcement. Formalized through Build Plan v3.5+ and codebase artifacts.

**The gap:** No bridge document exists that formally maps the relationship between these two expressions. Both reference the same 144-sphere structure independently, but:
- SHUGS uses resonance/frequency/harmonic language
- SNRS uses module/doctrine/invariant/routing language
- Neither explicitly acknowledges or cross-references the other
- A reader encountering both would not know they describe the same architecture

## §1 Bridge Requirements

The ORC-036 full document MUST address:

### §1.1 Mapping Table
For each of the 144 spheres, provide:
- SHUGS designation (resonance domain name, frequency, harmonic properties)
- SNRS designation (house, sphere ID, assigned modules)
- Equivalence statement (how the SHUGS description maps to the SNRS operational reality)

### §1.2 Layer Relationship
Clarify which expression applies at each decision point:
- **Routing decisions** → SNRS (operational)
- **Ontological classification** → Both (SHUGS provides the "why", SNRS provides the "how")
- **Module assignment** → SNRS (operational)
- **Sphere naming** → SHUGS (canonical names) with SNRS IDs (H01-S01 through H12-S12)
- **Governance** → SNRS (doctrines, invariants)
- **Philosophy/Purpose** → SHUGS (resonance, eternal properties)

### §1.3 Element 145 (The +1)
Both systems reference a 145th element:
- SHUGS: The "Unified Field" or "Source" element
- SNRS: Element 145 (ORC-012 TDD, technical design document)
- Bridge must clarify: same element, different descriptions? Or genuinely different concepts?

### §1.4 Conflict Resolution
If SHUGS and SNRS ever produce contradictory guidance for the same sphere:
- Which takes precedence?
- What is the escalation path?
- Does INV-0 (human consent) apply to metaphysical/resonance claims?
